# E-Commerce Backend

An ecommerce backend built with nodeJS, sequelize, mysql2, and other technologies. Users will be able to add and remove products, product categories, product tags, and product ids.

## Technologies Used

- Javascript
- Nodejs
- Express
- sequelize
- mysql2
- dotenv
